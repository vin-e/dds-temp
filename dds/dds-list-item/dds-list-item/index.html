dds-list-item

Version: 0.0.1
Path: packages/dds-list-item

The list item provides a item that can be rendered within `dds-dropdown`, `dds-select`, `dds-cascader` and other components. It provides the properties to specify if you want to make you list item an anchor link, button or regular div.

When specified as an anchor link (by providing the URL to the href) the implementer can also provide the `rel`, `target` and `download` attributes.

When specified as a button (via setting the button property to true) the implementer can also provide the type of button (i.e. button, reset or submit).

Regardless of setting the list item as an anchor link, button, or div you can set the list item to disabled or selected.

Properties
Property Attribute Description Type Default
button button Will render a button if `true` and href is not defined. boolean false
disabled disabled Removes interactivity if set to `true`. boolean false
download? download This attribute instructs browsers to download a URL instead of navigating to it. This attribute will pre-fill the file name for the download save prompt. string
href? href Contains a URL that will render an anchor tag. string
rel? rel Specifies the relationship of the target object to the link object. The value is a space-separated list of [link types](https://developer.mozilla.org/en-US/docs/Web/HTML/Link_types). string
selected selected Provides styling to imply that the user has selected the list item. boolean false
target? target Specifies where to display the linked URL (href). Allowed keywords: `"_blank"`, `"_self"`, `"_parent"`, `"_top"`. DdsListItemTarget.blank | DdsListItemTarget.parent | DdsListItemTarget.self | DdsListItemTarget.top
type type The type of button to be rendered. DdsListItemType.button | DdsListItemType.reset | DdsListItemType.submit DdsListItemType.button
Styles
Name Description
--item-color The color of the item text. Default: --color-grey-03);
--item-disabled-color The color of the item text when it is disabled. Default: --color-grey-06);
--item-hovered-background-color The background color when hovering and not disabled. Default: --color-grey-10);
--item-padding The default top and bottom padding. Default: 4px 0 6px;
--item-selected-border-left-color The border color when the item is selected. Default: --color-primary-05);
--item-selected-border-left-width The border width when the item is selected. Default: 3px;
--item-selected-border-radius The inside border radius for the border. Default: 0 4px 4px 0;
--item-selected-font-color The font color when the item is selected. Default: --color-primary-02);
