dds-filter

Version: 0.0.1
Path: packages/dds-filter

Filter component where you can specify:
- The columns used
- The filter used by setting the prop Filter
- Whether it is the first filter in the composite
- Listen for changes for filter on filterChanged
- Listen for events fired on clickAnd, clickOr, remove
